```
AVS8

│   ├── config.json - 
│   └── model_file.pth - 
│
└── vocoder/vocoder_models-universal--libri-tts--fullband-melgan
    ├── config.json - 
    ├── model_file.pth - 
    └── scale_stats.npy -
```
